Nix examples
============

Start python development with nix

::

    ./bin/nix-shell-py27

    or

    ./bin/nix-shell-py27

    or

    ./bin/nix-shell-py33

    or

    ./bin/nix-shell-py34

    or

    ./bin/nix-shell-pypy


After that use ``pip``, ``buildout`` as you're used to

::

     pip install pyramid --root `pwd`
